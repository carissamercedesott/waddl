# Security Policy

Waddl is an independent, early-stage open-source research project. Security
reports are welcome and appreciated.

## Reporting a vulnerability

Please **do not** open a public issue for security problems. Instead, email:

**carissaott0809@gmail.com**

Include what you found, how to reproduce it, and the potential impact. You'll
get an acknowledgement as soon as reasonably possible.

## Scope notes

- The `duckling` CLI runs locally and writes only to `~/.duckling`; it makes no
  network calls. Reports about unexpected data exfiltration are especially
  welcome.
- This is not a commercial product and has no formal SLA, but good-faith reports
  will be taken seriously and credited if you'd like.
