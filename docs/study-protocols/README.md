This directory holds the protocols for Waddl's studies. Each protocol is
published **before** data collection and describes the hypotheses,
design, measures, and analysis plan so results can be interpreted (and
replicated) honestly.

## Protocols

| Protocol | Study | Status |
| --- | --- | --- |
| [experiment-001-adaptive-hints.md](experiment-001-adaptive-hints.md) | Adaptive hints vs. direct solutions | Draft / pilot |

## What every protocol must contain

1. **Research question & hypotheses** — stated before data collection.
2. **Design** — within/between subjects, conditions, counterbalancing.
3. **Participants** — target N, recruitment, inclusion criteria, consent.
4. **Materials** — tasks, plugin configuration, instruments.
5. **Measures** — primary and secondary outcomes, operationalized.
6. **Procedure** — step by step, including timing.
7. **Analysis plan** — statistical tests, exclusions, stopping rule.
8. **Ethics & privacy** — consent, data handling, per [privacy.md](../privacy.md).

## Process

1. Draft the protocol here and open a PR for review/critique.
2. Pre-register (e.g. OSF) and link the registration.
3. Run a pilot; revise the protocol if needed (document changes).
4. Collect data per the frozen protocol.
5. Publish anonymized results and analysis under [`../../studies/`](../../studies/).
