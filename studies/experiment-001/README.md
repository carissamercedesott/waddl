# Experiment 001 — Adaptive hints vs. direct solutions

> Status: **protocol drafting / pilot.** No results yet.
>
> Research materials in this directory are licensed under CC BY 4.0
> (see [../../docs/LICENSE-CC-BY-4.0.md](../../docs/LICENSE-CC-BY-4.0.md)).

This is the home for the first Waddl study: does receiving **adaptive,
graduated hints** (vs. immediate full solutions) improve learning — as
measured by unaided transfer and self-reported understanding — when using
an AI coding assistant?
