## About this activity

Students in the club are invited to try **Duckling**, a free, open-source tool
that works inside the Claude Code app. Before giving a coding answer, it asks the
student to first predict what the code does, then shows what really happens — the
idea is to help them *understand*, not just get the answer. A **"Show Answer"**
option is always available, so students are never stuck.

This is an **informal feedback session**, not a research study. We want to know
if the tool is helpful and easy to use.

- **Voluntary:** Your student may choose not to take part, or stop at any time,
  with no effect on grades or club standing.
- **No data is collected about your student.** The tool runs entirely on the
  local computer; nothing about their code or activity is sent anywhere or
  stored by the project. The only thing gathered is an **anonymous** feedback
  survey with **no name** attached.
- **Duration:** About 20–30 minutes during a club meeting.
- **Who runs it:** The club advisor, [advisor name], with the tool created by an
  independent developer (not affiliated with the school).

Questions: contact the club advisor, or the project maintainer at
